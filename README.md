# Roger

Моя нейросеть, которую я улучшаю каждый день.

Copyright (c) 2025 Axolotl512, death_script. Лицензия: MIT.

ИИ, который подходит для встройки в робота и игр.

## Версии и архитектура

- **Quecto Roger 1 и ниже** — архитектура LP, .NET 8.0.
- **Quecto Roger 2 и выше** — архитектура MLP, .NET 9.0.
- **Yocto Roger 2.1** — архитектура MLP, .NET 9.0.

## Запуск  Yocto Roger 2.1 на Windows

1. Установите .NET 9.0.
2. Перейдите в по пути: `AI_Roger/AI_Roger/bin/Release/net9.0`
3. Запустите `AI_Roger.exe`.
4. > Или просто скачайте релизный бинарник
5. Отредактируйте файл знаний ([knowledge.know](https://raw.githubusercontent.com/Ivan125976/AI_Roger/refs/heads/v2.2-beta/Yocto_Roger/knowledge.know)) или создайте собственный (подробнее в wiki).

### Параметры обучения

- **LR (Learning Rate)** — чем меньше значение, тем аккуратнее обучение.
- **Проходы** — чем больше значение, тем лучше сеть усваивает информацию.
- **Файл знаний** — файл, на котором обучается нейросеть.
- **DropOut** (только в `AI_Roge.exe`) — защита от переобучения.
  Рекомендуемые значения: **5%** или **10%**.
  При использовании DropOut обычно нужен меньший LR и больше проходов.

## Запуск на Linux ARM (aarch64, AI_Roger)

1. В папке с файлами выполните:
   `./AI_Roger`
2. Отредактируйте файл знаний `[knowledge.know](https://raw.githubusercontent.com/Ivan125976/AI_Roger/refs/heads/v2.2-beta/Yocto_Roger/knowledge.know)` (подробнее в wiki).
